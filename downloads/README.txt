This is the downloads directory for ollama-lancache.

You can place any files here that you want to make available for download:
- Executable files (.exe, .msi, .deb, .rpm)
- Archive files (.zip, .tar.gz, .7z)
- Documentation (.pdf, .txt, .md)
- Configuration files
- Scripts

Files placed here will be available at:
http://your-server:8080/downloads/filename

The downloads page provides a web interface listing all available files
with their sizes and modification dates.

Security notes:
- Directory traversal is prevented (no ../ paths)
- Subdirectories are not supported for security
- Only regular files are served
